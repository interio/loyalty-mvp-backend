1) Create a tenant
POST /graphql
{
  "query": "mutation ($input: CreateTenantInput!) { createTenant(input: $input) { id name } }",
  "variables": { "input": { "name": "Tenant Alpha" } }
}
Capture tenantId.

2) Create two customers under that tenant
Customer A:
POST /graphql
{
  "query": "mutation ($input: CreateCustomerInput!) { createCustomer(input: $input) { id name tenantId pointsAccount { balance } } }",
  "variables": {
    "input": {
      "tenantId": "TENANT_ID",
      "name": "Blue Fox Bar",
      "contactEmail": "owner@bluefox.example",
      "externalId": "CUST-BLUE-001"
    }
  }
}

Customer B:
POST /graphql
{
  "query": "mutation ($input: CreateCustomerInput!) { createCustomer(input: $input) { id name tenantId pointsAccount { balance } } }",
  "variables": {
    "input": {
      "tenantId": "TENANT_ID",
      "name": "Green Garden Cafe",
      "contactEmail": "owner@greengarden.example",
      "externalId": "CUST-GREEN-001"
    }
  }
}
Capture customerId for each.

4) Transactions (ledger)
Use RedeemPoints for debits and ManualAdjustPoints for credits/debits. Examples per customer:
Customer A:
POST /graphql
{
  "query": "mutation ($input: RedeemPointsInput!) { redeemPoints(input: $input) { customerId balance updatedAt } }",
  "variables": { "input": { "customerId": "CUSTOMER_A_ID", "actorUserId": "USER_A1_ID", "amount": 50, "reason": "reward_redeem", "correlationId": "blue-redeem-1" } }
}

POST /graphql
{
  "query": "mutation ($input: ManualAdjustPointsInput!) { manualAdjustPoints(input: $input) { customerId balance updatedAt } }",
  "variables": { "input": { "customerId": "CUSTOMER_A_ID", "actorUserId": "USER_A2_ID", "amount": 100, "correlationId": "blue-adjust-1" } }
}
